Skybox
- left/right/back/front/bottom/top.jpg files are from https://learnopengl.com/Advanced-OpenGL/Cubemaps and I combined them into skybox.dds.
- texassemble.exe cube -w 2048 -h 2048 -o cubemap.dds right.jpg left.jpg top.jpg bottom.jpg front.jpg back.jpg -y -if CUBIC_DITHER_DIFFUSION -f BGRA
- texassemble.exe cube -w 2048 -h 2048 -o cubemap.dds posx.jpg negx.jpg posy.jpg negy.jpg posz.jpg negz.jpg -y -if CUBIC_DITHER_DIFFUSION
- texassemble.exe can be downloaded from here https://github.com/Microsoft/DirectXTex/wiki/Texassemble
- skybox*.dds files were generated by filtering skybox.dds file using IBLBaker. https://github.com/derkreature/IBLBaker

Other *.dds files 
- are from Microsoft DirectX-Graphics-Samples at https://github.com/microsoft/DirectX-Graphics-Samples.
